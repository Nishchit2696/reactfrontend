Labeling and Identification Technology: There are many Identification code or tag that can be used to identify the object. Most popular code is bar-code which maximum used throughout the world for the identification of serial number for the object. For scanning the bar-code need line-of-sign (LoS) which have relatively short distance. Manufacture cost relatively less to the company, since it only required special software to print the bar-code. But bar-code has limitation of storing data. Despite limitation, it improve the speed of tracking inventory then old traditional identification procedure. As time pass bar-code evolved towards Quick Response (QR) or BiDimensional (BiDi) code. It can store more than 1800 characters data of the inventory which can be read by the smart phone camera, thus, reduce the overall cost of the inventory tracking system. QR code can be read up to the distance depends on the size of the QRcode size. Since both bar-code and QR code has a limited reading distance and need LoS between scanner and the code. As labels are evolved to the sophisticated labels. The next step of inventory tractability system is based on RFID. As such kind of label has a distance of several centimeter to several tons of meter. In additional, RFID can store certain data of the inventory. And some modern RFID is capable of sensing the temperature, acceleration of light. The next evolution of the tag is called smart tag which is more complex then RFID tag. Smart tag is more complex that can be storing functionality like event detection, embed IOT sensor that can collect the data from the surrounding that is relevant for the inventory. As tags are getting smarter which make identification tractability system autonomous and effective for the ware house, that can protect the inventory and can communicate with multiple device for more accurate record keeping.
